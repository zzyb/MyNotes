# 袋子、队列、堆栈

### Bag

袋子是一个不支持移除项的集合----其目的是为客户提供收集项的能力。

### FIFO queues

先进先出队列是一个基于先进先出 (FIFO) 策略的集合类型。

### Pushdown stack

下压栈是一个基于后进先出 (LIFO) 策略的集合类型。
